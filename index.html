LLM 伦理决策倾向性研究
大学生研究计划 [A-] · 大创计划 [A] · ICWSM 2026
  • 研究动机: 大语言模型在面对伦理困境时的决策是否存在系统性偏见？关闭式与开源模型表现有何差异？
  • 评测框架: 构建多维度伦理困境题库，涵盖功利主义 vs 道义论、生命价值权衡、公平与效率等经典伦理张力场景
  • 实验规模: 对 9 种主流模型（GPT-4, Claude, Qwen, DeepSeek 等）执行 50,400 次并发 API 调用，使用 Python asyncio + 限流策略控制并发
  • 分析维度: 伦理偏好优先级排序、跨模型稳定性、偏好聚类分析、温度参数影响、提示词敏感性
  • 核心发现: 国际上首次发现大模型在伦理困境中存在稳定的系统性偏见——闭源模型倾向功利主义，部分开源模型表现出更强的道义论倾向
  • 技术栈: Python, OpenAI API, Anthropic API, asyncio, matplotlib, scipy, scikit-learn
CCL2026-Eval 古诗词赏析 · 大模型微调
天池AI大模型系列赛 · 296 队 Top 10%
  • 模型选型: 选用 Qwen2.5-7B-instruct 作为基座模型，平衡推理能力与微调成本
  • 微调架构: 设计两阶段 LoRA——第一阶段用 23,000 条通用古诗词数据粗调领域知识；第二阶段用 1,800 条精标任务数据针对性微调六大任务（字词释义、典故识别、情感推理、意境分析、修辞鉴赏、文化背景）
  • 数据工程: 从古籍数据库爬取原始诗词文本，清洗格式、去重、切分，使用规则 + 人工抽样验证数据质量
  • 消融实验: 引入外部大模型生成额外题库及 CoT 思维链微调数据，但消融实验显示反而降低性能（过拟合到外部模型风格），果断舍弃该方案
  • 技术栈: PyTorch, Transformers, PEFT/LoRA, vLLM 推理加速, Weights & Biases 实验追踪
傅里叶变换光谱 · 物理实验
全国大学生物理实验竞赛 国家级三等奖 · 《物理实验》期刊
  • 实验设计: 搭建 Michelson 干涉光路，通过傅里叶变换光谱技术测量黑体辐射光谱，验证维恩位移定律
  • 算法实现: 使用 MATLAB 编写 FFT 去噪、相位校正、光谱重建算法，处理干涉图数据
  • 论文发表: 作为第二作者，论文发表于《物理实验》2024 年第 44 卷第 8 期
  • 技术栈: MATLAB, 光学实验, 信号处理, LaTeX
