# 常见因子类型

> 因子 = 生成交易信号的原始数据或计算方法。回答"买什么、何时买"的问题。

---

## 一、动量因子 (Momentum)

**核心思想**: 过去涨的继续涨，过去跌的继续跌。

| 类型 | 计算方式 | 典型参数 |
|------|----------|----------|
| 价格动量 | 过去 N 期收益率 | N=1, 5, 21, 252 日 |
| 截面动量 | 横截面排名 | 行业内/全市场排名 |
| 残差动量 | 剔除市场和行业 Beta 后的残差收益 | CAPM/Fama-French 残差 |
| 时间序列动量 | 自身过去收益符号 | Moskowitz et al. (2012) |

**为什么有效**: 行为金融（羊群效应、锚定偏差）、机构资金流入的持续性。

**风险**: 动量崩溃（momentum crash）——市场急剧反转时动量策略大幅回撤。

**我的理解**: 类似物理的惯性。但金融市场没有牛顿第一定律——动量可以突然消失。MadEvolve 论文里用一分钟 BTCUSD，动量因子的预测能力很弱（因为市场近乎有效）。

---

## 二、反转因子 (Reversal / Mean Reversion)

**核心思想**: 涨多了会跌，跌多了会涨。

| 类型 | 计算方式 |
|------|----------|
| 短期反转 | 过去 1-5 日收益率的相反数 |
| 长期反转 | 过去 3-5 年收益率的相反数 |
| 日内反转 | 开盘到盘中最高/最低的反弹 |

**为什么有效**: 流动性提供者的补偿、过度反应后的修正、市场微观结构噪音。

**与动量的关系**: 短期（日/周）通常是反转，中期（月/年）是动量。这个切换点本身就是研究课题。

---

## 三、波动率因子 (Volatility)

**核心思想**: 低波动率的资产长期收益更高（低波动异象）。

| 类型 | 计算方式 |
|------|----------|
| 已实现波动率 | 过去 N 日收益率的标准差 |
| 特质波动率 | Fama-French 残差的标准差 |
| Beta | 对市场收益的回归系数 |
| 下行波动率 | 只计算负收益的标准差 |

**为什么有效**: 杠杆约束（低波动资产需要加杠杆才能达到目标收益，但很多机构不能加杠杆）、彩票偏好（投资者喜欢高波动"彩票"股票，推高价格压低未来收益）。

---

## 四、流动性因子 (Liquidity)

**核心思想**: 流动性差的资产需要更高的预期收益作为补偿。

| 类型 | 计算方式 |
|------|----------|
| Amihud 非流动性 | |日收益率| / 日成交额 |
| 买卖价差 | (Ask - Bid) / Mid |
| 换手率 | 成交量 / 流通股数 |
| 市场深度 | 订单簿上各价位的挂单量 |

**我的理解**: MadEvolve 里用限价单策略（被动挂单），本质上是提供流动性来吃手续费补贴。这和流动性因子直接相关——流动性越差的市场，提供流动性的收益越高，但风险也越大。

---

## 五、情绪因子 (Sentiment)

**核心思想**: 市场参与者的情绪影响价格。

| 类型 | 数据来源 |
|------|----------|
| 新闻情绪 | NLP 分析财经新闻 |
| 社交媒体 | Twitter/Reddit 情感分析 |
| 分析师一致预期 | 盈利预测修正 |
| 期权隐含信息 | Put/Call 比率、VIX |
| 资金流向 | 大单/小单资金流 |

---

## 六、质量因子 (Quality)

**核心思想**: 财务质量好的公司长期表现更好。

| 维度 | 指标 |
|------|------|
| 盈利能力 | ROE, ROA, 毛利率 |
| 盈利稳定性 | 盈利增长率的标准差 |
| 资本结构 | 资产负债率, 流动比率 |
| 成长性 | 盈利增长率, 营收增长率 |

**注意**: 质量因子主要用于股票，加密货币市场不适用（没有财务报表）。

---

## 七、价值因子 (Value)

**核心思想**: 便宜的资产未来收益更高。

| 类型 | 指标 |
|------|------|
| 市盈率 | P/E |
| 市净率 | P/B |
| 企业价值/EBITDA | EV/EBITDA |
| 股息率 | Dividend Yield |

---

## 因子研究的核心问题

1. **因子有效性检验**: IC (信息系数)、ICIR (IC 的信息比率)、分层回测
2. **因子衰减**: 因子发现后多久会失效？市场会学习
3. **因子拥挤**: 太多人用同一个因子 → 收益被套利掉
4. **过拟合**: 数据挖掘了 N 个因子，总有几个"看起来显著"——需要多重检验校正
