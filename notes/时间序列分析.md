# 时间序列分析

> 基于 Box, Jenkins, Reinsel & Ljung *Time Series Analysis: Forecasting and Control*（第 5 版，Wiley, 2016）的整理笔记。

**时间序列**是一组按时间先后顺序排列的观测值 $\{z_t, t = 1, 2, \ldots, n\}$。其核心特征在于**序列依赖性**：相邻观测值之间存在统计关联，该关联既携带被研究系统的动态信息，又构成统计推断中不可忽视的数据结构约束。

**时间序列分析**的目标可以归纳为三个层次：描述（分解系统的结构性成分）、建模（以参数化随机过程刻画数据的生成机制）、预测（基于模型外推未来观测值及其不确定区间）。

### 模型构建的基本思想

#### 迭代建模三阶段

这是 Box-Jenkins 方法论的核心：

```
 ┌──────────────┐
 │ 1. 识别      │
 │(Identification)│
 └──────┬───────┘
        │ 提出暂定模型
        ▼
 ┌──────────────┐
 │ 2. 估计      │
 │(Estimation)  │
 └──────┬───────┘
        │ 获得参数值
        ▼
 ┌──────────────┐     不充分
 │ 3. 诊断检验   │──────────→ 返回阶段 1
 │(Diagnostic   │
 │ Checking)    │
 └──────┬───────┘
        │ 模型充分
        ▼
      预测 / 应用
```

#### 简约性原则 (Principle of Parsimony)

在同等拟合精度下，参数最少的模型更优。Box 的核心理念：「所有模型都是错误的，但有些是有用的」（*All models are wrong, but some are useful.*）

## 2. 平稳过程的自相关函数与频谱

### 弱平稳的三个条件

1. $E[Z_t] = \mu$（常数均值）
2. $\text{Var}[Z_t] = \sigma^2$（常数方差）
3. $\text{Cov}[Z_t, Z_{t+k}] = \gamma_k$（自协方差仅依赖于滞后 $k$）

### 自协方差与自相关

**自协方差**：$\gamma_k = \text{Cov}(Z_t, Z_{t+k}) = E[(Z_t - \mu)(Z_{t+k} - \mu)]$

**自相关函数 (ACF)**：$\rho_k = \gamma_k / \gamma_0$

### 样本估计

$$
\hat{\rho}_k = r_k = \frac{\sum_{t=1}^{n-k} (z_t - \bar{z})(z_{t+k} - \bar{z})}{\sum_{t=1}^{n} (z_t - \bar{z})^2}
$$

> 分母使用 $n$（而非 $n - k$），保证自协方差矩阵的正定性。

对于白噪声过程，样本自相关的标准误约为 $1/\sqrt{n}$，95% 置信带约为 $\pm 2/\sqrt{n}$。

### 谱密度函数

谱密度函数是自协方差函数的 Fourier 变换：

$$
f(\omega) = \frac{1}{2\pi} \sum_{k=-\infty}^{\infty} \gamma_k e^{-i\omega k} = \frac{1}{2\pi} \left[ \gamma_0 + 2\sum_{k=1}^{\infty} \gamma_k \cos(\omega k) \right]
$$

- 白噪声的谱密度为常数（所有频率贡献相等）
- AR(1) 正参数的谱在低频处较高（缓慢趋势样波动）
- MA(1) 的谱可能在高频或低频出现低谷

## 3. 线性平稳模型

### 一般线性过程（Wold 分解）

$$
\tilde{z}_t = a_t + \psi_1 a_{t-1} + \psi_2 a_{t-2} + \cdots = \psi(B) a_t
$$

平稳条件：$\sum |\psi_j| < \infty$。

### AR($p$) 过程

$$
\tilde{z}_t = \phi_1 \tilde{z}_{t-1} + \cdots + \phi_p \tilde{z}_{t-p} + a_t
$$

即 $\phi(B)\tilde{z}_t = a_t$，其中 $\phi(B) = 1 - \phi_1 B - \cdots - \phi_p B^p$。

**平稳条件**：$\phi(z) = 0$ 的所有根 $|z| > 1$。

**AR(1)**：$\tilde{z}_t = \phi_1 \tilde{z}_{t-1} + a_t$，$|\phi_1| < 1$。ACF：$\rho_k = \phi_1^{|k|}$（指数衰减）。

**AR(2)**：平稳域为 $\phi_1 + \phi_2 < 1$，$\phi_2 - \phi_1 < 1$，$-1 < \phi_2 < 1$。当特征根为共轭复数时，ACF 呈衰减正弦波。

**Yule-Walker 方程**：

$$
\gamma_k = \phi_1 \gamma_{k-1} + \cdots + \phi_p \gamma_{k-p}, \quad k \ge 1
$$

### 偏自相关函数 (PACF)

对 AR($p$) 过程，PACF 在 $k = p$ 后**截尾**。标准误：$\text{SE}(\hat{\phi}_{kk}) \approx 1 / \sqrt{n}$。

### MA($q$) 过程

$$
\tilde{z}_t = a_t + \theta_1 a_{t-1} + \cdots + \theta_q a_{t-q} = \theta(B) a_t
$$

**可逆条件**：$\theta(z) = 0$ 的所有根 $|z| > 1$。

**MA(1)**：$\tilde{z}_t = a_t + \theta_1 a_{t-1}$，$|\theta_1| < 1$。ACF：$\rho_1 = \theta_1 / (1 + \theta_1^2)$，$\rho_k = 0 \ (k \ge 2)$。

### AR 与 MA 的对偶性

| 属性 | AR($p$) | MA($q$) |
|------|---------|---------|
| ACF | 拖尾 | $k > q$ 截尾 |
| PACF | $k > p$ 截尾 | 拖尾 |
| $\psi$ 权重 | $\phi(B)\psi(B) = 1$ | $\psi_j = \theta_j$（有限） |
| $\pi$ 权重 | $\pi_j = \phi_j$（有限） | $\theta(B)\pi(B) = 1$ |

### ARMA($p, q$) 过程

$$
\phi(B)\tilde{z}_t = \theta(B)a_t
$$

**ARMA(1,1)**：$\tilde{z}_t = \phi_1 \tilde{z}_{t-1} + a_t + \theta_1 a_{t-1}$

- 平稳：$|\phi_1| < 1$；可逆：$|\theta_1| < 1$
- ACF 从滞后 2 起指数衰减：$\rho_k = \phi_1^{k-1}\rho_1 \ (k \ge 2)$

## 4. 线性非平稳模型——ARIMA

### ARIMA($p, d, q$)

若 $\nabla^d z_t = (1 - B)^d z_t$ 为平稳可逆 ARMA($p, q$)，则 $z_t$ 服从 ARIMA($p, d, q$)：

$$
\phi(B)(1 - B)^d z_t = \theta_0 + \theta(B)a_t
$$

$d$ 通常取 0 或 1，$d = 2$ 已属罕见。

### 三种等价表示

**① 差分方程形式**（预测计算）：

$$
z_t = \varphi_1 z_{t-1} + \cdots + \varphi_{p+d} z_{t-p-d} - \theta_1 a_{t-1} - \cdots - \theta_q a_{t-q} + a_t
$$

**② $\psi$ 权重形式**（预测误差方差）：

$$
z_t = \sum_{j=0}^{\infty} \psi_j a_{t-j}
$$

$l$ 步预测误差方差：$\text{Var}[e_t(l)] = \sigma_a^2(1 + \sum_{j=1}^{l-1} \psi_j^2)$

对 $d > 0$，$\sum |\psi_j|$ 不收敛——预测区间无界增长。

**③ $\pi$ 权重形式**（反转形式）：

$$
z_t = \sum_{j=1}^{\infty} \pi_j z_{t-j} + a_t
$$

对于 $d \ge 1$：$\sum \pi_j = 1$。$\pi$ 权重总是收敛的。

### 确定性漂移与 IMA 过程

IMA(0,1,1)：$\nabla z_t = a_t + \theta_1 a_{t-1}$

加入漂移：$\nabla z_t = \theta_0 + a_t + \theta_1 a_{t-1}$

当 $\theta_1 = 1 - \alpha$ 时，IMA(0,1,1) 的预测等价于简单指数平滑。

## 5. 预测

### MMSE 预测

$\hat{z}_t(l) = E[z_{t+l} \mid z_t, z_{t-1}, \ldots]$

### 预测函数的形式

- $d = 0$（平稳）：预测随 $l \to \infty$ 收敛于过程均值（均值回复）
- $d = 1$：预测函数为初始值决定的水平线加漂移——"永记"上一次观测的信息
- $d = 2$：预测函数为二次函数

### 预测概率区间

$(1 - \alpha)$ 概率区间：

$$
\hat{z}_t(l) \pm u_{\alpha/2} \left(1 + \sum_{j=1}^{l-1} \hat{\psi}_j^2\right)^{1/2} \hat{\sigma}_a
$$

### 预测的更新

$$
\hat{z}_{t+1}(l) = \hat{z}_t(l+1) + \psi_l [z_{t+1} - \hat{z}_t(1)]
$$

其中 $z_{t+1} - \hat{z}_t(1) = a_{t+1}$ 为一步预测误差（新息）。

## 6. 模型识别

### ACF / PACF 模式识别表

| 模型 | ACF 行为 | PACF 行为 |
|------|----------|-----------|
| **白噪声** | 所有滞后 $\approx 0$ | 同左 |
| **AR($p$)** | 拖尾（指数衰减或衰减正弦波） | $k = p$ 后**截尾** |
| **MA($q$)** | $k = q$ 后**截尾** | 拖尾（指数衰减） |
| **ARMA($p, q$)** | 拖尾 | 拖尾 |
| **非平稳** | 衰减极慢，$\hat{\rho}_1$ 常接近 1 | $\hat{\phi}_{11}$ 接近 1 |

### 参数的初始估计

- **AR($p$)**：将样本自相关代入 Yule-Walker 方程求解
- **MA($q$)**：对自协方差方程非线性求解，始终选择可逆解
- **ARMA($p, q$)**：先用 AR 方法估计 $\phi$，再从残差估计 $\theta$

## 7. 参数估计

### 似然函数

在正态性假设下，精确对数似然：

$$
\log L(\phi, \theta, \sigma_a^2 \mid \mathbf{z}) = -\frac{n}{2}\log(2\pi\sigma_a^2) - \frac{1}{2\sigma_a^2} S(\phi, \theta)
$$

- **条件似然**：固定前 $p$ 个 $z$ 和前 $q$ 个 $a = 0$
- **无条件似然**：通过回测 (backcasting) 估计前样本残差

### 非线性估计

MA 参数非线性地进入模型，需 Gauss-Newton / Marquardt 算法进行数值优化。

---

## 8. 模型诊断检验

### Ljung-Box 检验

$$
Q(m) = n(n + 2) \sum_{k=1}^{m} \frac{\hat{r}_k^2(\hat{a})}{n - k} \sim \chi^2(m - p - q)
$$

若 $p$ 值 > 0.05，模型通过诊断。

### 过度拟合

有意增加一个 AR 或 MA 参数——若额外参数不显著，则确认当前模型已充分。

## 9. 季节时间序列分析

### SARIMA$(p, d, q)(P, D, Q)_s$

$$
\phi(B)\Phi(B^s)(1 - B)^d(1 - B^s)^D z_t = \theta(B)\Theta(B^s)a_t
$$

航空公司模型 SARIMA$(0,1,1)(0,1,1)_{12}$（仅两个参数）是简约性的经典展示。

### 含 ARIMA 误差的回归

$$
z_t = \mathbf{x}_t'\boldsymbol{\beta} + n_t, \quad n_t \sim \text{ARIMA}(p, d, q)
$$

## 10. 扩展主题

### 单位根检验

ADF 检验：$\nabla z_t = \gamma z_{t-1} + \sum \delta_i \nabla z_{t-i} + a_t$，检验 $H_0: \gamma = 0$（单位根）。

### ARCH / GARCH

**GARCH(1,1)**（Bollerslev, 1986）：

$$
\sigma_t^2 = \omega + \alpha a_{t-1}^2 + \beta \sigma_{t-1}^2
$$

平稳条件：$\alpha + \beta < 1$。$\alpha + \beta$ 越接近 1，波动率冲击越持久。

**EGARCH**：引入 $\gamma$ 捕捉杠杆效应（负冲击比正冲击更剧烈地推高波动率）。

### 长记忆过程——ARFIMA

ARFIMA($p, d, q$)，$d \in (-0.5, 0.5)$：

$$
\phi(B)(1 - B)^d z_t = \theta(B) a_t
$$

当 $0 < d < 0.5$ 时，序列平稳但 ACF 以双曲率衰减（$\rho_k \sim C k^{2d-1}$）。

## 11. 传递函数模型

$$
y_t = \frac{\omega(B)}{\delta(B)} B^b x_t + n_t
$$

### 预白化与互相关函数 (CCF)

1. 对输入 $x_t$ 拟合 ARIMA，获得白噪声化滤波器
2. 对输出 $y_t$ 施加同一滤波器
3. 计算 CCF：$\rho_{\alpha\beta}(k)$
4. CCF 的形状直接对应冲击响应权重 $v_k$，由此读取传递函数阶数 $r, s, b$

## 12. 干预分析与异常值检测

### 干预分析

| 干预类型 | 传递函数 | 响应特征 |
|---------|---------|---------|
| 突变—永久 | $\omega_0 I_t$ | 立即跳变至新水平并永久保持 |
| 渐变—永久 | $\frac{\omega_0}{1 - \delta_1 B} I_t$ | 逐步收敛至 $\omega_0 / (1 - \delta_1)$ |
| 脉冲—衰减 | $\frac{\omega_0}{1 - \delta_1 B} P_t$ | 指数衰减回原水平 |

### 四种异常值类型

| 类型 | 影响方式 |
|------|---------|
| **加性异常值 (AO)** | 仅影响单个观测 |
| **新息异常值 (IO)** | 通过 ARIMA 动态持续影响后续序列 |
| **水平漂移 (LS)** | 水平突变并永久保持 |
| **暂时变化 (TC)** | 水平突变后按指数衰减 |

## 13. 多元时间序列分析

### VAR($p$)

$$
\mathbf{z}_t = \boldsymbol{\Phi}_1 \mathbf{z}_{t-1} + \cdots + \boldsymbol{\Phi}_p \mathbf{z}_{t-p} + \mathbf{a}_t
$$

平稳条件：$|\mathbf{I} - \boldsymbol{\Phi}_1 z - \cdots - \boldsymbol{\Phi}_p z^p| = 0$ 的所有根在单位圆外。

### 协整与 VECM

当各分量非平稳但存在 $r$ 个平稳的线性组合时：

$$
\nabla \mathbf{z}_t = \boldsymbol{\Pi} \mathbf{z}_{t-1} + \sum_{j=1}^{p-1} \boldsymbol{\Gamma}_j \nabla \mathbf{z}_{t-j} + \mathbf{a}_t
$$

其中 $\boldsymbol{\Pi} = \boldsymbol{\alpha}\boldsymbol{\beta}'$（$\boldsymbol{\alpha}$ 为调整速度，$\boldsymbol{\beta}$ 为协整向量）。

---

## 手动演算：IMA(0,1,1)

以 $\nabla z_t = a_t - 0.7 a_{t-1}$，$\sigma_a^2 = 1.0$ 为例。

观测：$z = [10.0, 12.5, 11.8, 14.2, 13.9, 15.5]$

### $\psi$ 权重
- $\psi_0 = 1$，$\psi_1 = 0.3$，$\psi_j = 0.3$（$j \ge 1$）——不收敛（$d > 0$）

### $\pi$ 权重
- $\pi_j = 0.3 \times 0.7^{j-1}$——指数收敛，$\sum \pi_j = 1$ ✓

### 残差递推与预测

$\hat{z}_6(1) = z_6 - 0.7 a_6 = 15.5 - 0.7 \times 1.997 = 14.102$

$l \ge 2$ 时预测值恒等于 $\hat{z}_6(1)$——IMA(0,1,1) 预测函数在一步之后不再更新。

## 常见误区

| ❌ 误区 | ✅ 正解 |
|--------|--------|
| "差分越多越好" | 过度差分会引入负自相关，应以最小 $d$ 达到平稳 |
| "ACF 统计显著 = 实际重要" | 关注模式（截尾 vs 拖尾）而非单个值的显著性 |
| "一个序列只有一个正确模型" | 多个 ARMA 结构可能拟合几乎相同，选最简约的 |
| "预测区间越窄越好" | 区间反映固有不可预测性，不应人为缩窄 |

## 术语速查

| 术语 | 简要含义 |
|------|---------|
| 白噪声 | 零均值、恒定方差、序列无关的随机过程 |
| Wold 分解 | 协方差平稳过程唯一分解为白噪声单边移动平均 |
| 平稳性 | 均值、方差恒定且自协方差仅依赖于滞后 |
| 可逆性 | MA 过程可唯一表示为 AR($\infty$) 的条件 |
| 新息 | $a_t = z_t - \hat{z}_{t-1}(1)$ |
| 简约性原则 | 同等拟合下参数最少的模型最优 |
| $\psi$ 权重 | $z_t$ 表示为纯冲击加权和的系数（预测误差方差） |
| $\pi$ 权重 | $z_t$ 表示为纯过去值加当前冲击的系数（反转预测） |
| 传递函数 | 输入序列到输出序列的线性动态滤波器 |
| 预白化 | 对输入施加滤波器使其变为白噪声后再做互相关 |
| 协整 | 非平稳序列的线性组合变为平稳 |
| 杠杆效应 | 负面冲击对波动率的放大超过同等正面冲击 |
| 波动率聚集 | 大幅波动后倾向跟随大幅波动的时变方差特征 |
